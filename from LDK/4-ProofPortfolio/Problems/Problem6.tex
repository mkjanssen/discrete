%Using proof by csases/contrapositive/biconditionals (combining methods)
	
\noindent\textbf{Conjecture 6A.}   For all integers $a$, $4\mid a$ if and only if there exists an integer $b$ such that $a = 1 + (-1)^b(2b-1)$.\\  %Both directions true.  Existence proof one way, cases (t even/odd) the other.

\noindent\textbf{Conjecture 6B.} For all integers $k$, $3 \nmid k^2+2$ if and only if $k$ is not prime or $k=3$.\\  %Contrapositive and cases. One direction false.

\noindent\textbf{Conjecture 6C.} For all integers $x$ and $y$, $xy\equiv 0 \pmod{3}$ if and only if $x\equiv 0 \pmod{3}$ or $y\equiv 0\pmod{3}$.
