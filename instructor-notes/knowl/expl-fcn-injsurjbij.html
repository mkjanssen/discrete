Exploration 4.3.1.

Consider the functions below. Are they injective, surjective, neither, or bijective? Explain.

  1. \(f: \N\to \N\) given by the rule \(f(n) = 2n\text{.}\)
  2. \(f: \N\to \set{0,2,4,6,\ldots}\) given by the rule \(f(n) = 2n\text{.}\)
  3. \(f: \set{1,2,3,4,5}\to \set{1,2,3,4}\) given by the two-line notation
    \begin{equation*} f = \begin{pmatrix}1 \amp 2 \amp 3 \amp 4 \amp 5 \\ 2 \amp 1 \amp 3 \amp 1\amp 4 \end{pmatrix}, \end{equation*}
    where the number in the first row is understood to map to the number below it.
  4. The birthday function described in Exploration 4.2.1.
  5. The sum of divisors function described in Exploration 4.2.2.
  6. The function \(f : \N\to\Z\) given by
    \begin{equation*} f(n) = \begin{cases} n/2 \amp \text{ if } n \text{ is even} \\ -(n+1)/2 \amp \text{ if } n \text{ is odd}\end{cases}. \end{equation*}
