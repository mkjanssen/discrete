Exploration 5.3.3.

Let's consider bit strings of length 6. We'll denote the set of such strings as \(\mathbf{B}^6\text{.}\) Further, define the weight of a bit string as the number of 1's in the string (equivalently, the sum of the bits in the string). We'll denote the set of \(n\)-bit strings with weight \(k\) by \(\mathbf{B}^n_k\text{.}\)

  1. How many weight four bit strings are there? That is, what is \(\card{\mathbf{B}^6_4}\text{?}\)
  2. Calculate \(\card{\mathbf{B}^6_3}\) and \(\card{\mathbf{B}^6_2}\text{.}\)
  3. How are these questions related to the subset question?
