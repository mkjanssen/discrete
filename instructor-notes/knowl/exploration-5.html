Exploration 2.1.3.

This exercise is practice with checking if a conditional statement is true. Consider the following conditional statement:

If I win the lottery, every student in this class gets an A.
In this conditional statement, what is the hypothesis and what is the conclusion? Which of the following circumstances make the conditional statement bolded above false?

  1. I win the lottery and every student in this class gets an A.
  2. I win the lottery and not every student in this class gets an A.
  3. I don’t win the lottery and everyone in this class gets an A.
  4. I don’t win the lottery and not everyone in this class gets an A.
