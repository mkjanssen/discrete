Here's another counting problem involving sets. Given a set \(A\text{,}\) recall that the power set of \(A\text{,}\) denoted \(\pow(A)\text{,}\) is the set of all subsets of \(A\text{.}\) Can we predict ahead of time how many subsets a set will have? That is, given a set \(A\text{,}\) can we calculate \(\card{\pow(A)}\) if we know \(\card{A}\text{?}\)
