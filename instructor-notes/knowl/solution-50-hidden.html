A proof by contradiction will be used. So we assume the proposition is false. This means that there exists a real number \(x\) such that \(x(1-x) \gt \frac{1}{4}\text{.}\) If we multiply both sides of this inequality by \(4\text{,}\) we obtain \(4x(1-x) > 1\text{.}\) However, if we let \(x=3\text{,}\) we then see that

\begin{align*} 4x (1-x) \amp \gt 1\\ 4\cdot 3(1-3) \amp\gt 1 \\ -12 \amp \gt 1. \end{align*}

The last inequality is clearly a contradiction and so we have proved the proposition.
