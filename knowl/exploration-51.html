Exploration 11.3.2.

Write clear proofs of each of the following statements. Then swap yours with a friend's and offer feedback on their correctness and clarity.

  1. For all integers \(n\text{,}\) if \(n\) is odd, then \(n^2\) is odd.
  2. For all odd integers \(m,n\text{,}\) \(m+n\) is even.
  3. Let \(x\) be an integer. Then \(x\) is odd if and only if there exists an integer \(\ell\) for which \(x = 2\ell -1\text{.}\)
  4. Let \(a,b,c\in\Z\) with \(a|b\) and \(a|c\text{.}\) Then \(a|(b+c)\text{.}\)
  5. Let \(a,b,c\in\Z\) with \(a|b\) and \(a|c\text{.}\) Then \(a|bc\text{.}\)
  6. Let \(a,b,c\in\Z\) with \(a|b\) and \(b|c\text{.}\) Then \(a|c\text{.}\)
