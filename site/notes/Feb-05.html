Section 4.3 Feb. 5: Functions

Motivating Questions.

Here are some questions we'll consider.

  1. What is the image of an element/set under a function?
  2. What is the complete inverse image of an element in the codomain of a function?
  3. What is the inverse image of the subset of the codomain of a function?

Subsection 4.3.1 Outline

Today's material comes from the last part of Section 0.4 of the book. You're encouraged to read it before coming to class.

Subsection 4.3.1.1 More on Injections, Surjections, and Bijections

Exploration 4.3.1.

Consider the functions below. Are they injective, surjective, neither, or bijective? Explain.

  1. \(f: \N\to \N\) given by the rule \(f(n) = 2n\text{.}\)
  2. \(f: \N\to \set{0,2,4,6,\ldots}\) given by the rule \(f(n) = 2n\text{.}\)
  3. \(f: \set{1,2,3,4,5}\to \set{1,2,3,4}\) given by the two-line notation
    \begin{equation*} f = \begin{pmatrix}1 \amp 2 \amp 3 \amp 4 \amp 5 \\ 2 \amp 1 \amp 3 \amp 1\amp 4 \end{pmatrix}, \end{equation*}
    where the number in the first row is understood to map to the number below it.
  4. The birthday function described in Exploration 4.2.1.
  5. The sum of divisors function described in Exploration 4.2.2.
  6. The function \(f : \N\to\Z\) given by
    \begin{equation*} f(n) = \begin{cases} n/2 \amp \text{ if } n \text{ is even} \\ -(n+1)/2 \amp \text{ if } n \text{ is odd}\end{cases}. \end{equation*}
Exploration 4.3.2. A very discrete math question.

Using the two-line notation described above, write out all functions \(f: \set{1,2,3}\to \set{a,b}\text{.}\)

  1. How many such functions are there?
  2. How many are injective?
  3. How many are surjective?
  4. How many are bijective?

Subsubsection 4.3.1.1 Recursive Definitions of Functions

We've so far seen that we can define functions by listing ordered pairs, giving rules, drawing arrow diagrams, and the like. Another important method for defining functions (especially in computer science) is with a recursive definition.

Recursively defined functions are often easier to create from a “real world” problem, because they describe how the values of the functions are changing. However, this comes with a price. It is harder to calculate the image of a single input, since you need to know the images of other (previous) elements in the domain.

Recursively Defined Functions.

For a function \(f:\N \to \N\text{,}\) a recursive definition consists of an initial condition together with a recurrence relation. The initial condition is the explicitly given value of \(f(0)\text{.}\) The recurrence relation is a formula for \(f(n+1)\) in terms for \(f(n)\) (and possibly \(n\) itself).

Exploration 4.3.3.

Consider the recurrence \(f : \N\to\N\) given by \(f(n+1) = 3f(n)\text{.}\) Note that this is not enough to define a function since we don't have an initial condition. For each initial condition below, find the value of \(f(4)\text{.}\)

  1. \(\displaystyle f(0)=0\)
  2. \(\displaystyle f(0)=1\)
  3. \(\displaystyle f(0)=2\)
  4. \(\displaystyle f(0)=100\)

Subsubsection 4.3.1.2 Image

Let \(f : X\to Y\) be a function. The image of \(x\in X\) under \(f\) is the value \(y\in Y\) for which \(f(x) = y\text{.}\) This extends to entire subsets of the domain. Given \(A\subseteq X\text{,}\) the image of \(A\) under \(f\) is the set

\begin{equation*} f(A) = \setof{f(x)}{x\in A}. \end{equation*}
Exploration 4.3.4.

Consider the function \(f : \N\times\N \to \N\) given by \(f((a,b)) = a+b\text{.}\) Given \(A = \setof{(a,b)\in \N\times\N}{a,b\le 10}\text{,}\) find (with justification) \(f(A)\text{.}\)

Example 4.3.1.

Suppose \(f : X\to Y\) and \(A,B\subseteq X\text{.}\) Is it true that \(f(A\cup B) = f(A)\cup f(B)\text{?}\)

Subsubsection 4.3.1.3 Inverse Images (Preimages)

If images tell us where functions send elements and sets of the codomain, then inverse images (sometimes called preimages) describe which objects in the domain get sent to particular objects (elements, subsets) of the codomain.

I differ from our book in preferring the term “preimage”; the inverse image of sets and elements in the codomain can be calculated even for functions which do not have inverses, and I find the double meaning of the word “inverse” annoying.

Given a subset \(B\subseteq Y\text{,}\) the inverse image of \(B\) under \(f\) the set

\begin{equation*} f^{-1}(B) = \setof{x\in X}{f(x)\in B}. \end{equation*}

If \(B = \set{y}\) for some \(y\in Y\text{,}\) we often write \(f^{-1}(\set{y})\) as \(f^{-1}(y)\text{;}\) it is the complete inverse image of \(y\) under \(f\) (but again, \(f^{-1}\) need not be a well-defined function).

Exploration 4.3.5.

Consider again the function from Exploration 4.3.4.

  1. Find \(f^{-1}(3)\) and \(f^{-1}(\set{0,1,2,3})\text{.}\)
  2. Give geometric descriptions of \(f^{-1}(n)\) and \(f^{-1}(\set{0,1,2,\ldots,n})\) for any \(n\ge 1\text{.}\)
  3. Calculate \(|f^{-1}(8)\) and \(|f^{-1}(\set{0,1,\ldots,8})\text{.}\)
Question 4.3.2.

Suppose \(f : X\to Y\) is a function and \(1\in Y\text{.}\) What can you say about \(f^{-1}(1)\) if you that:

  1. \(f\) is injective? Explain.
  2. \(f\) is surjective? Explain.
  3. \(f\) is bijective? Explain.
Challenge.

Let \(f: X\to Y\) be a function with \(A\subseteq X\) and \(B\subseteq Y\text{.}\)

  1. Is \(f^{-1}(f(A))=A\text{?}\) Always? Sometimes? Never? Explain.
  2. Is \(f(f^{-1}(B)) = B\text{?}\) Always? Sometimes? Never? Explain.
  3. If one or both of the above do not always hold, is there something else you can say? Will equality always hold for particular types of functions? Is there some other relationship other than equality that would always hold? Explore.
As you consider the statements above, you may want to build one or two representative examples to test. But be careful! If your examples are too similar, you may be led down an erroneous path.
