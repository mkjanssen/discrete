Section 12.2 Mar. 31: Other Types of Proof

Motivating Questions.

Here are some questions we'll consider.

Subsection 12.2.1 Outline

Today's material comes from Section 3.2 of the book. You're encouraged to read it before coming to class.

Subsubsection 12.2.1.1 Proof by Contrapositive

Example 12.2.1.

Is the statement “for all integers \(n\text{,}\) if \(n^2\) is even, then \(n\) is even” true?

Example 12.2.2.

Prove: for all integers \(a\) and \(b\text{,}\) if \(a + b\) is odd, then \(a\) is odd or \(b\) is odd.

Subsubsection 12.2.1.2 Proof by Cases

It often feels like we don't quite have enough information to complete a proof. It is possible to make additional assumptions locally in cases, provided that our cases are exhaustive.

Example 12.2.3.

If \(n\in\Z\text{,}\) then \(n^2+n\) is even.

Exploration 12.2.1.

For all real numbers \(a,b\text{,}\) if \(ab=0\text{,}\) then \(a=0\) or \(b=0\text{.}\)

Hint
First assume that \(a=0\text{.}\) What are the remaining case(s)?
Definition 12.2.4.

For any \(x\in\R\text{,}\) the absolute value of \(x\), denoted \(|x|\text{,}\) is defined to be

\begin{equation*} |x| = \begin{cases} x \amp \text{if } x\ge 0\\ -x \amp \text{if } x\lt 0\end{cases} \end{equation*}
Exploration 12.2.2.

Let \(a\) be a positive real number. Prove that \(|x| = a\) if and only if \(x=a\) or \(x=-a\text{.}\)

Subsubsection 12.2.1.3 Miscellany

Exploration 12.2.3.

Consider the statement, “For each odd negative integer \(x\) that is a multiple of 6, we have \(2|(x+1)\)”. Is this true or false? Justify.

Exploration 12.2.4.

Consider the statement “For all integers \(a,b\text{,}\) if \(a+b\) is odd, then \(a\) is odd or \(b\) is odd”. Is this statement true? What about the converse?
