Exploration 6.2.3.

Consider the set \(A = \set{a,b,c,d,e,f,g,h}\text{.}\)

  1. How many ways are there to permute all the elements of \(A\text{?}\)
  2. How many 7-letter “words” (that is, strings of letters) can you make from the elements of \(A\text{?}\) Assume no repeated letters.
  3. How many 6-letter words can you make from the elements of \(A\text{?}\) Assume no repeated letters.
  4. Suppose now that \(X\) is a set of cardinality \(n\text{.}\) How many ways are there to order \(k\le n\) distinct objects from \(X\text{?}\) Conjecture a formula.
  5. Suppose \(f: S\to T\) is a function with \(\card{S} = k\) and \(\card{T} = n\text{.}\) How many such functions \(f\) are injective?
