Suppose first that \(d(a) = 1\text{.}\) In this case, \(a\) has exactly one neighbor, \(x\text{.}\) It is possible that \(x = b\) or \(x\ne b\text{;}\) we'll check both possibilities. First, let \(G' = G\setminus a\text{.}\) Notice that \(d(x)\) drops by 1, while all other vertices have the same degree as before. Also note that \(G\) has \(m\) edges and is connected.
