Let \(X\) be a set and \(\sim\) a relation on \(X\text{,}\) so that if \(x,y\in X\) and \((x,y)\in \sim\text{,}\) we may write the more conventional \(x\sim y\text{.}\)

  1. We say that \(\sim\) is reflexive if whenever \(x\in X\text{,}\) \(x\sim x\text{.}\)
  2. We say that \(\sim\) is symmetric if whenever \(x,y\in X\) and \(x\sim y\text{,}\) then \(y\sim x\text{.}\)
  3. We say that \(\sim\) is transitive if whenever \(x,y,z\in X\) such that \(x\sim y\) and \(y\sim z\text{,}\) then \(x\sim z\text{.}\)
