Proof

Begin with our graph \(G\) and identify a spanning tree \(T\) of \(G\text{.}\) Let \(v_T, e_T, f_T\) denote the corresponding number of each object of \(T\text{.}\)

Observe that each time that we remove an edge of \(G\) not in \(T\text{,}\) two faces merge. When we break all cycles of \(G\) (leaving us with a tree), we have a single face. Thus, we remove \(f-1\) edges from \(G\) to achieve \(T\text{,}\) i.e.,

\begin{equation*} e - (f-1) = e_T\text{.} \end{equation*}

Since \(T\) is a tree, we know that \(e_T = v_T - 1\text{,}\) so \(v_T - e_T = 1\text{.}\) The spanning tree has exactly one face, so \(f_T = 1\text{,}\) and adding these two together give us \(v_T - e_T + f_T = 2\text{.}\) Moreover, since \(T\) is a spanning tree, \(v_T = v\text{.}\) Substituting this and EquationÂ  into \(v_T - e_T + f_T = 2\) we have \(v - (e-(f-1))+f_T = 2\text{,}\) which simplifies to our desired result as \(f_T = 1\text{.}\)
