The question we ask is: how many 3-element subsets of \(A= \set{1,2,\ldots, 10}\) are there? One answer (based on previous work) is \(\binom{10}{3}\text{.}\)

For the other answer, consider a three element subset \(\set{a,b,c}\) with \(a \lt b \lt c\text{.}\) We count based on values for \(b\text{.}\)

  • If \(b=2\text{,}\) then there is one choice for \(a\) and 8 choices for \(c\text{,}\) so \(1\cdot 8\) choices in total.
  • If \(b=3\text{,}\) there there are 2 choicesa for \(a\) and \(7\) choices for \(c\text{,}\) so \(2\cdot 7\) choices in total.
  • Etc. etc.

The identity is established.
