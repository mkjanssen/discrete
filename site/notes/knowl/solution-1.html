Solution 6.2.1.1.
  1. If the bottom chip must be red or blue, there are two choices there. Then there are five choices for the next (top) chip, so \(2\cdot 5 = 10\text{.}\)
  2. For each of the 10 stacks from the previous part, there are three choices for the new top chip, so \(3\cdot 10 = 30\) choices in total.

  3. I read the problem with order mattering, e.g., RBY is a different stack than YBR. First, choose the three distinct colors from the five in \(\binom{5}{3}\) ways. For each choice of three colors, there are \(3\cdot 2\cdot 2 = 6\) ways to order them. This leaves 60 total stacks.

    If we're building a stack of four chips, there are 5 choices of which chip to leave out and \(4!\) ways to order the colors once they're selected, for \(5! = 120\) orderings.

  4. If we only want to choose the colors, there are \(\binom{5}{3}\) and \(\binom{5}{4}\) ways to do so.
