Solution 7.3.5.1.

This is the same as before. There are 15 units to assign to five boxes. So, \(\binom{15+5-1}{5-1} = 3876\text{.}\)

The second one requires each variable to be at least 1, so instead we assign 10 units to 5 boxes: \(\binom{10+5-1}{5-1} = 1001\text{.}\)

Last, we assign 2 units to each variable, leaving 5 to be assigned: \(\binom{5+5-1}{5-1} = 126\text{.}\)

Of course, if each variable needs to be at least 3, there are \(\binom{5-1}{5-1} = 1\) ways to accomplish this.
