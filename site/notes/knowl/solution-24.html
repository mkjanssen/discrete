  3. This one is trickier! Notice that the terms themselves are each \(a_n = 3 + \sum_{k=1}^n (2k-1), n\ge 1\text{.}\) Obviously, 20 3's is 60, so we need the sum
    \begin{equation*} \sum\limits_{n=1}^{20} \left(\sum\limits_{k=1}^n 2k-1\right). \end{equation*}
    Consider the sum \(S_n = \sum\limits_{k=1}^n 2k-1 = 1 + 3 + 5 + \cdots + 2n-1\text{.}\) Then
    \begin{equation*} 2S_n = \underbrace{2n+2n+2n+\cdots+2n}_{\text{n times}} = 2n^2, \end{equation*}
    so \(S_n = n^2\text{.}\) Thus, we want to find \(60 + \sum\limits_{n=1}^{20} n^2 = 60 + 2870 = 2930\text{.}\)
