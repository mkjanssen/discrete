Solution 9.3.4.1.
  1. \(P(3) = P(2) + T_3 = 4 + 6 = 10\text{,}\) \(P(4) = 10 + T_4 = 20\text{,}\) \(P(5) = 20 + T_5 = 20 + 15 = 35\text{.}\)
  2. The sequence is \(1, 4, 10, 20, 35, \ldots\text{.}\) The first difference is \(3, 6, 10, 15\ldots\text{,}\) so the second is \(3, 4, 5\text{,}\) an arithmetic sequence. Thus, the sequence is \(\Delta^3\)-constant and will be described by a cubic, \(P_n = a n^3 + b n^2 + cn + d\text{.}\)

  3. We can solve the system of equations:

    \begin{align*} a + b + c + d \amp= 1\\ 2^3 a + 2^2 b + 2c + 1 \amp=4\\ 3^3 a + 3^2 b + 3c + 1 \amp=10\\ 4^3 a + 4^2 b + 4c + 1 \amp=20 \end{align*}

    We get \(P_n = \frac{1}{6}n^3 + \frac{1}{2}n^2 + \frac{1}{3}n\text{.}\)

  4. \(\displaystyle P_{15} = 680.\)
