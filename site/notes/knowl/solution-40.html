Solution 10.3.2.1.

Anything amount \(n\ge 8\text{.}\) Observe that we can create postage in the amounts of 8, 9, 10, 11, and 12 cents. Assume that for all \(12 \le k \lt n\) cents we can create postage in that amount. Note that this means we can create postage in the amount of \(n-3\) cents; add a 3 cent stamp to make postage in the amount of \(n\) cents.
